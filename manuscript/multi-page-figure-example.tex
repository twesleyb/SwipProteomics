% What do you do if your figure and its caption are too large to fit on a single
% page? A solution from: https://latex.org/forum/viewtopic.php?t=5881

%--- Preamble ---%

\documentclass{article}

\usepackage{lipsum} % For dummy text.
\usepackage{graphicx} % For graphics.
\usepackage{caption} % For captions that span multple pages.

\graphicspath{ {./images/} }

% Multi-page-figure caption format.
\DeclareCaptionLabelFormat{adja-page}{\hrulefill\\#1 #2 \emph{(Continued from previous page)}}

%--- Main ---%

\begin{document}

% Intro
\section{Intro}

A large figure (\ref{fig:test}) with its caption on the subsequent page. This
solution is not perfect as the figure is not centered properly. I read online
that if the image is larger that pagewidth, then it won't be centered.

\begin{figure}[!hb]
	\begin{center}
	\captionsetup{labelformat=empty}
	\noindent\includegraphics[width=0.5\paperwidth,keepaspectratio]{Figure_01}
	\caption{} % An empty caption is required.
	\end{center}
\end{figure}

\begin{figure}[!t]
	\captionsetup{labelformat=adja-page}
	\ContinuedFloat
	\caption{\protect\lipsum[1]}
	\label{fig:test}
\end{figure}

% The text goes on...
\lipsum[1-2]

\end{document}
